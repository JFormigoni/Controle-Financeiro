# Camada de Componentes (`src/components`)

Componentes React (Server e Client Components) reutilizáveis da camada de
apresentação: layout responsivo, navegação, formulários, gráficos e elementos de
UI. Os layouts seguem os breakpoints de responsividade (`md` = 768px,
`lg` = 1024px) definidos em `tailwind.config.ts` (Req. 13.1-13.3).
